# HackLib

Game/Process hacking library built in rust.

## Features

- [X] Project hooking.
- [X] Memory:
  - [X] Read
  - [X] Write
  - [ ] Offsets.
  - [ ] Paging & Lookup.
- [ ] Render:
  - [ ] GL/DX Context hooking.
  - [ ] Primitive drawing (Line, rect)
  - [ ] Text drawing.
  - [ ] Color support.
  - [ ] Sprite drawing.
  - [ ] Mouse and keyboard dispatching.
- [ ] Injection:
  - [ ] Shell code.
  - [ ] DLLs.
- [ ] Threads:
  - [ ] Pause/resume.
  - [ ] Spawning.
